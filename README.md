# std

Types and utility functions brokered with [jsrepo](https://jsrepo.dev). If I have wrote it twice it's probably here.

## Setup

Add as a repo in config:

```bash
jsrepo init @ieedan/std
```

then add your blocks:

```bash
jsrepo add ts/result
```
